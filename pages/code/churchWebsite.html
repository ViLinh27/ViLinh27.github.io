Church Website

This is a fictional church. The website is based on existing websites of religious institutions.

Here is a preview of what the live site looks like. Visit here to see more of it.

I wanted to make a simple website based on a fictional organization as a sort of ongoing starter project for the purpose of learning frontend web development. This is purely for educational purposes, but I didn’t want to use an existing church’s websiten to redesign, so I used a fictional one from a story I’m working on. I hadn’t touched any markup language in years so the organization of the code is ist still pretty haphazard.

Concerning the design and information architectuer of the website, I heavily based this site off existing churches’ websites. One was Our Lady of Gudalupe in Helotes Texas. The site can be visited here and is previewed below.

The other was Our Lady of Atonement (also in Helotes Texas). Ths site can be visited here and is previewed below.

This is a continuous work in progress project technically but I have the basic sketletons of the main home page. There are flaws that need to be fixed like responsiveness, but this was a good chance to get back into web development after several years of not touching it.

Some of these screens are still bare-bones and links don't all work, because I have a questionable habit of working on several projects at once.
