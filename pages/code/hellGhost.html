HellGhost

A simple endless runner made in Unity

I learned a bit of C# in high school and undergrad in some intro programming classes. It's a good starting language because it has the structure of most Object Oriented languages while having syntax that's easy enough to understand. Unity was a complex game engine that I did not have much exposure to, but I saw that you could 2D games, which seemed more accessible to me. Thanks to searching through several youtube tutorials, I found a good one that went through the steps quite well. You can find the channel here.

Blackthornprod, the channel I used for youtube reference for this game even has a full playlist to show you how to make an endless runner game that can be visited here.

The assets were made in photoshop. If interested in playing the game or seeing more details, my itchio page features it. It can be visited here .
